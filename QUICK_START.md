# 快速开始 - 5分钟上手

## 🚀 立即部署

```bash
git add .
git commit -m "feat: 完整版 Globo Pre-Order"
git push
```

## ✅ 部署后配置

### 1. 访问管理后台
```
https://shopmall.dpdns.org/admin?shop=your-shop.myshopify.com
```

### 2. 开启自动预购
进入 **设置** → 开启 **"自动预购"** → 保存

### 3. 测试
- 将任何商品库存设为 0
- 访问商品页面
- 应自动显示 "Pre-Order Now"

## 📋 核心功能

| 模式 | 说明 |
|------|------|
| 🤖 自动 | 库存=0自动启用 |
| ✋ 手动 | 商家手动控制 |
| 📦 批量 | 一次处理多个 |
