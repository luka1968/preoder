# 🛍️ 商家使用指南 - PreOrder Pro (Globo 模式)

**给商家的说明：如何在你的 Shopify 商店使用预购功能**

---

## ✅ 安装步骤（3 分钟）

### **第 1 步：安装 App**

1. 访问 Shopify App Store 或你的安装链接
2. 点击"添加应用"
3. 确认权限并安装

---

### **第 2 步：启用 App Embed**

安装完成后，会自动跳转到主题编辑器：

1. 在左侧找到 **"App embeds"** 区域
2. 找到 **"PreOrder Pro - 预购插件"**
3. 打开开关 ✅（变为绿色）
4. 点击右上角 **"保存"**

**截图示例：**
```
┌─────────────────────────────────┐
│ App embeds                      │
├─────────────────────────────────┤
│ ☑️ PreOrder Pro - 预购插件      │ ← 开启这个
│   🚀 Globo 模式：自动加入购物车 │
│                                 │
│ ☐ 其他 App...                   │
└─────────────────────────────────┘
```

---

### **第 3 步：配置预购产品**

对于每个想要预购的产品：

1. **进入 Shopify Admin**
2. **Products** → 选择产品
3. **在 Inventory 部分：**
   - 将库存数量设为 `0`
   - 找到 "When out of stock"
   - 选择 **"Continue selling when out of stock"** ✅
4. 点击 **保存**

**为什么要这样设置？**
- 库存为 0：触发预购功能
- Continue selling：允许客户购买缺货商品

---

## 🎉 完成！现在测试一下

### **测试步骤：**

1. **访问你的在线商店**
2. **打开设置为缺货的产品页面**
3. **你应该看到：**
   - ✅ 产品图片上有 "🔥 預售 Pre-Order" 徽章
   - ✅ "🛒 立即预订 Pre-Order Now" 按钮（代替原来的 Add to Cart）
   - ✅ 预计发货日期说明（可选）

4. **点击预购按钮：**
   - 商品会自动加入购物车
   - 自动跳转到结账页面
   - 客户可以立即完成支付

5. **查看订单：**
   - 在 Shopify Admin → Orders 可以看到订单
   - 在 App 后台也可以管理预购订单

---

## ⚙️ 自定义设置（可选）

### **修改预购按钮样式**

1. 进入 Shopify Admin
2. Online Store → Themes → Customize
3. 点击左侧 **App embeds** 中的 **PreOrder Pro**
4. 可以修改：
   - ✏️ 按钮文字（默认："立即预订 Pre-Order Now"）
   - ✏️ 徽章文字（默认："预售 Pre-Order"）
   - 🐛 调试模式（测试时开启）

## 📋 常见使用场景

### **场景 1：新品预售**
```
1. 创建新产品
2. 库存设为 0
3. 开启 "Continue selling"
4. 客户可以预购新品
5. 货到后发货
```

### **场景 2：爆款补货**
```
1. 产品卖光了（库存 = 0）
2. 自动显示预购按钮
3. 客户继续下单
4. 补货后统一发货
```

### **场景 3：限量发售**
```
1. 设置库存为 0
2. 只允许预购
3. 控制发售数量
4. 按预购顺序发货
```

## 📊 管理预购订单

### **在 App 后台查看：**

1. 打开 PreOrder Pro App
2. 进入 **订单管理** 页面
3. 可以看到所有预购订单：
   - 客户信息
   - 产品信息
   - 支付状态
   - 发货状态

### **在 Shopify Admin 查看：**

所有预购订单都是正常的 Shopify 订单：
- Orders → 可以看到所有订单
- 预购订单会有特殊标记
- 可以正常处理发货、退款等

---

## 🚚 如何发货

### **当商品到货后：**

1. **在 Shopify Admin**
2. **Orders → 找到预购订单**
3. **点击 "Fulfill items"**
4. **填写物流信息**
5. **客户会收到发货通知**

就这么简单！预购订单和普通订单的发货流程完全一样。

## 🐛 故障排查

### **问题 1：预购按钮没有显示**

**检查：**
- [ ] App Embed 是否已启用？
- [ ] 产品库存是否为 0？
- [ ] 是否开启了 "Continue selling when out of stock"？

**解决：**
1. 进入主题编辑器
2. 检查 App embeds 中 PreOrder Pro 是否开启
3. 确认产品设置正确

---

### **问题 2：点击按钮没反应**

**检查：**
- [ ] 浏览器是否阻止了弹窗？
- [ ] 网络连接是否正常？

**解决：**
1. 按 F12 打开浏览器控制台
2. 查看是否有错误信息
3. 联系技术支持并提供错误截图

---

### **问题 3：添加到购物车失败**

**最常见原因：**
没有开启 "Continue selling when out of stock"

**解决：**
1. Shopify Admin → Products → 选择产品
2. Inventory → When out of stock
3. 选择 "Continue selling when out of stock"
4. 保存
